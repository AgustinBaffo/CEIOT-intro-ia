# Clase 1 - NumPy para AI
En ésta clase se abarcarán las características principales de la API de NumPy así como una justificación ténica de su uso. El temario es el siguiente:
* Performance
* Arrays
* Vistas vs. Copias
* Slicing, Indexing
* Dimensiones
* TensorSensor
* Broadcasting
